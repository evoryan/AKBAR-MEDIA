Pesan "Akses ditolak, token tidak ditemukan" berarti URL yang Anda buka di browser memerlukan login (misalnya URL API seperti /api/odc atau lainnya).

Untuk menjalankan perbaikan database, pastikan Anda membuka URL berikut di browser:
http://103.253.245.25:4500/api/fix-db

Jika error tersebut muncul saat Anda menggunakan Aplikasi Android:
Silakan Logout dari aplikasi, lalu Login kembali agar aplikasi mendapatkan token baru dari server.
